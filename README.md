# Summer Training: Genome Analysis 2026

## Overview

This repository documents my hands-on Summer Training in Genomics and Metagenomics conducted at the Department of Biosciences, Graphic Era University.

The objective of this repository is to document every practical performed during the training, including installation, bioinformatics pipelines, commands, outputs, troubleshooting, and learning outcomes.

---

## Training Modules

### Module 1: Genomics

- Linux & WSL Setup
- Ubuntu Installation
- Miniconda Installation
- NGS Quality Control
- Genome Assembly
- Assembly Evaluation
- Gene Prediction
- Genome Annotation

### Module 2: Metagenomics

- Quality Control
- Metagenome Assembly
- Binning
- Taxonomic Analysis
- Functional Annotation
