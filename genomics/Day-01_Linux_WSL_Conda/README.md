## Objective

The objective of this practical was to prepare a complete Linux-based bioinformatics environment using Windows Subsystem for Linux (WSL), Ubuntu, and Miniconda. This setup forms the foundation for performing genome analysis workflows throughout the training.

---

## Topics Covered

- Introduction to Linux
- Installing WSL
- Installing Ubuntu
- Installing Miniconda
- Creating Conda environments
- Installing bioinformatics software
- Checking software versions
- Running FastP
- Linux file management

## Learning Outcomes

After completing this practical, I was able to

- Install Ubuntu on Windows using WSL
- Configure Miniconda
- Create isolated Conda environments
- Install bioinformatics packages
- Execute Linux commands
- Perform basic NGS preprocessing using FastP
