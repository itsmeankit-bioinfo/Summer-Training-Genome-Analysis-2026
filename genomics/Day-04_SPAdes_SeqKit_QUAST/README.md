# Day 04 — Genome Assembly using SPAdes, SeqKit & QUAST

> **Training Module:** Genomics  
> **Duration:** Day 4  
> **Tools Used:** SPAdes, SeqKit, QUAST, Ubuntu (WSL), Conda

---

# Objective

The objective of this practical was to assemble sequencing reads into longer contiguous sequences (contigs) using SPAdes, evaluate assembly quality using QUAST, and analyze assembled sequences using SeqKit.

---

# Topics Covered

- Introduction to Genome Assembly
- Contigs and Scaffolds
- De novo Assembly
- SPAdes Genome Assembler
- SeqKit
- QUAST
- Assembly Statistics
- N50 and L50
- Assembly Quality Evaluation

---

# Learning Outcomes

After completing this practical, I was able to:

- Perform genome assembly using SPAdes.
- Understand the concept of contigs.
- Evaluate assembly quality using QUAST.
- Analyze assembled sequences using SeqKit.
- Interpret assembly statistics.

---

# Why This Practical Matters

Genome assembly converts millions of short sequencing reads into longer DNA sequences that can be used for genome annotation and downstream biological analyses.

Assembly quality directly affects annotation accuracy, comparative genomics, and functional analysis.
