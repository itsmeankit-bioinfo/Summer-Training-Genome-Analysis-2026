# Day 05 — Reference Genome Analysis using TYGS & QUAST

> **Training Module:** Genomics  
> **Duration:** Day 5  
> **Tools Used:** TYGS, QUAST, Ubuntu (WSL)

---

# Objective

The objective of this practical was to compare an assembled genome with a reference genome, evaluate assembly quality, and understand genome identification using the Type (Strain) Genome Server (TYGS).

---

# Topics Covered

- Reference Genome
- Genome Identification
- Type Strain Genome Server (TYGS)
- Assembly Evaluation
- Reference-based QUAST Analysis
- Genome Similarity

---

# Learning Outcomes

After completing this practical, I was able to:

- Understand the concept of a reference genome.
- Perform assembly evaluation against a reference genome.
- Learn the purpose of TYGS.
- Interpret QUAST reports generated using a reference genome.
- Understand genome comparison and species identification.

---

# Why This Practical Matters

Reference genome comparison is essential for validating newly assembled genomes and identifying unknown microbial isolates. It provides confidence in assembly quality and helps determine evolutionary relationships.
