# Case Study

# Genome Identification Using Reference Genomes

## Background

A bacterial genome was assembled using sequencing reads.

The identity of the organism was unknown.

---

## Problem

Without comparison to a reference genome, it is difficult to determine:

- Species identity
- Assembly completeness
- Assembly accuracy

---

## Solution

The assembled genome was compared with a reference genome using QUAST and uploaded to TYGS for bacterial identification.

Assembly statistics and genome similarity were analyzed.

---

## Real-world Example

Clinical microbiology laboratories compare newly sequenced bacterial genomes with reference genomes to identify pathogens and monitor disease outbreaks.

---

## Conclusion

Reference-based genome analysis improves assembly validation and enables reliable species identification.
