# Notes

# What is a Reference Genome?

A reference genome is a well-characterized DNA sequence used as a standard for comparison with newly sequenced genomes.

It serves as a template for genome assembly validation, annotation, and comparative analysis.

---

# Why are Reference Genomes Important?

Reference genomes help researchers:

- Identify unknown organisms
- Compare genome assemblies
- Detect mutations
- Perform comparative genomics
- Validate assembly quality

---

# What is TYGS?

The Type (Strain) Genome Server (TYGS) is an online platform used for whole-genome-based bacterial taxonomy.

It compares uploaded genomes with type strain genomes and helps identify bacterial species.

---

# Applications of TYGS

- Species identification
- Genome comparison
- Taxonomic classification
- Phylogenetic analysis

---

# Reference-Based Assembly Evaluation

Reference-based evaluation compares an assembled genome against a known reference genome.

Benefits include:

- Detecting missing regions
- Identifying misassemblies
- Measuring genome completeness
- Improving assembly confidence

---

# QUAST with Reference Genome

When a reference genome is provided, QUAST calculates additional metrics including:

- Genome fraction
- Misassemblies
- Mismatches
- Insertions and deletions
- Alignment statistics

---

# Genome Similarity

Genome similarity measures how closely two genomes are related.

Closely related genomes usually belong to the same species or genus.

---

# Key Takeaways

- Reference genomes improve assembly evaluation.
- TYGS is widely used for bacterial genome identification.
- QUAST provides detailed comparison against a reference genome.
- Reference-based analysis is an essential step in comparative genomics.
