# Objective

The objective of this practical was to predict protein-coding genes from an assembled bacterial genome using Prodigal and understand the concept of Open Reading Frames (ORFs) in genome annotation.

# Learning Outcomes

After completing this practical, I was able to:

- Understand gene prediction.
- Explain the concept of ORFs.
- Predict coding sequences using Prodigal.
- Interpret Prodigal output files.
- Prepare assembled genomes for genome annotation.

---

# Why This Practical Matters

Genome assembly only reconstructs DNA sequences.

Gene prediction identifies biologically meaningful regions that encode proteins.

Without gene prediction, genome annotation cannot be performed.
