> **Training Module:** Genomics  
> **Duration:** Day 7  
> **Tools Used:** Prokka, Prodigal, Barrnap, Aragorn, BLAST+, HMMER

# Objective

The objective of this practical was to perform rapid genome annotation using Prokka and understand how protein-coding genes, rRNA genes, tRNA genes, and other genomic features are identified and functionally annotated.

---

# Learning Outcomes

After completing this practical, I was able to:

- Understand genome annotation.
- Perform bacterial genome annotation using Prokka.
- Interpret Prokka output files.
- Differentiate between structural and functional annotation.
- Prepare annotated genomes for downstream biological analysis.

---

# Why This Practical Matters

Genome assembly alone only provides DNA sequences.

Genome annotation identifies genes, proteins, RNAs, and their biological functions, making the assembled genome biologically meaningful.
