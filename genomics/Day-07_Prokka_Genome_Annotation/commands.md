# Commands Used

## Step 1 — Activate Conda Environment

```bash
conda activate bioinfo
```

Purpose

Activates the Conda environment containing Prokka and its dependencies.

---

## Step 2 — Check Current Directory

```bash
pwd
```

Displays the current working directory.

---

## Step 3 — List Files

```bash
ls
```

Verifies that the assembled genome (FASTA) is available.

---

## Step 4 — Run Prokka

```bash
prokka contigs.fasta --outdir prokka_output --prefix sample
```

Purpose

Performs automatic genome annotation on the assembled genome.

---

## Step 5 — View Output Files

```bash
ls prokka_output
```

Purpose

Displays all annotation files generated by Prokka.

---

## Step 6 — Count Predicted Proteins

```bash
grep ">" prokka_output/sample.faa | wc -l
```

Purpose

Counts the total number of predicted proteins.

---

## Step 7 — View Annotation Summary

```bash
cat prokka_output/sample.tsv | head
```

Purpose

Displays the first few annotated genes.

---
