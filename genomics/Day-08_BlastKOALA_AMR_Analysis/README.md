# Day 08 — Functional Annotation using BlastKOALA & Antimicrobial Resistance (AMR) Analysis

> **Training Module:** Genomics  
> **Duration:** Day 8  
> **Tools Used:** BlastKOALA, KEGG, AMRFinderPlus (or AMR database used during training)

---

# Objective

The objective of this practical was to assign biological functions to predicted genes using BlastKOALA and identify antimicrobial resistance (AMR) genes within a bacterial genome.

---

# Learning Outcomes

After completing this practical, I was able to:

- Understand functional genome annotation.
- Assign KEGG Orthology (KO) identifiers.
- Interpret metabolic pathways.
- Identify antimicrobial resistance genes.
- Analyze biological functions of predicted proteins.

---

# Why This Practical Matters

Genome annotation predicts genes, but functional annotation explains what those genes do.

Functional annotation and AMR analysis are essential for understanding bacterial metabolism, pathogenicity, and antibiotic resistance.
