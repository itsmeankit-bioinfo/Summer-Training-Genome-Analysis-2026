# Common Errors & Troubleshooting

## Error 1

BlastKOALA upload failed.

### Cause

Incorrect input file format.

### Solution

Upload the protein FASTA (.faa) file generated by Prokka.

---

## Error 2

No KO assignments found.

### Cause

Proteins have low similarity to KEGG database entries.

### Solution

Verify protein prediction quality and database selection.

---

## Error 3

AMR genes not detected.

### Cause

The organism may not contain known resistance genes or the database may not include relevant entries.

### Solution

Confirm the correct input file and database version.

---

## Error 4

KEGG pathways incomplete.

### Cause

Some proteins could not be assigned KO identifiers.

### Solution

Review annotation quality and repeat functional analysis if required.

---

## Best Practices

- Use high-quality protein sequences.
- Verify annotation before pathway analysis.
- Use updated reference databases.
