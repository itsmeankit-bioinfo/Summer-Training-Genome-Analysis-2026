# Day 11 — MetaQUAST & SAMtools

> **Training Module:** Metagenomics
> **Duration:** Day 11
> **Tools Used:** MetaQUAST, SAMtools

---

# Objective

The objective of this practical was to evaluate metagenome assembly quality using MetaQUAST and perform alignment file processing using SAMtools. This included converting SAM files into BAM format, sorting alignments, and generating BAM index files for downstream metagenomic analysis.

# Learning Outcomes

After completing this practical, I was able to:

- Evaluate metagenome assemblies using MetaQUAST.
- Understand assembly quality metrics.
- Convert SAM files to BAM format.
- Sort BAM files.
- Generate BAM index files.
- Prepare alignment files for downstream analyses.

---

# Why This Practical Matters

Assembly quality directly affects downstream analyses such as genome binning and functional annotation.

SAMtools provides efficient processing of alignment files, making large sequencing datasets easier to manage and analyze.
