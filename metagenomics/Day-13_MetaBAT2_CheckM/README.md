# Day 13 — Genome Binning using MetaBAT2 & MAG Quality Assessment using CheckM

> **Training Module:** Metagenomics
> **Duration:** Day 13
> **Tools Used:** MetaBAT2, jgi_summarize_bam_contig_depths, CheckM

---

# Objective

The objective of this practical was to reconstruct microbial genomes from assembled metagenomic contigs using MetaBAT2 and evaluate the quality of the resulting Metagenome-Assembled Genomes (MAGs) using CheckM.

# Learning Outcomes

After completing this practical, I was able to:

- Understand genome binning.
- Generate coverage depth files.
- Perform automated genome binning using MetaBAT2.
- Evaluate MAG quality using CheckM.
- Interpret completeness and contamination scores.

---

# Why This Practical Matters

Metagenomic assemblies contain contigs from many microorganisms.

Genome binning reconstructs individual microbial genomes, while CheckM evaluates their quality before downstream biological analysis.
