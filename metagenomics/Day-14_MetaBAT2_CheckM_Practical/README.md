# Day 14 — Genome Binning using MetaBAT2 & Genome Quality Assessment using CheckM

> **Training Module:** Metagenomics
> **Duration:** Day 14
> **Tools Used:** MetaBAT2, CheckM

---

# Objective

The objective of this practical was to perform genome binning using MetaBAT2, organize the generated genome bins, and assess the quality of reconstructed Metagenome-Assembled Genomes (MAGs) using CheckM.

---

# Topics Covered

- MetaBAT2 Practical
- Genome Binning
- Contig Binning
- MAG Generation
- CheckM Installation
- CheckM Lineage Workflow
- Genome Completeness
- Genome Contamination
- Quality Assessment

---

# Learning Outcomes

After completing this practical, I was able to:

- Install and configure MetaBAT2.
- Perform genome binning from assembled contigs.
- Organize generated genome bins.
- Install and use CheckM.
- Assess MAG quality using completeness and contamination metrics.

---

# Why This Practical Matters

Genome binning reconstructs microbial genomes from complex metagenomic datasets.

CheckM evaluates the quality of reconstructed genomes, helping researchers identify high-quality MAGs suitable for downstream biological analysis.
